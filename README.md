# Nikki App

わたしの日記の書き方にマッチする、オリジナルの日記ウェブアプリケーション

## Features

- 投稿は小さい単位で行う
  - その単位を仮にBlockと呼ぶことにする
  - `/@:user_name/blocks/:block_id`
  - Blockは、𝕏のようなマイクロブログの1投稿、Tumblrの1投稿(Link、Quote、Image、…)のようなもの
  - Blockを投稿してもタイムラインのような場所には表示されない
- Blockを束ねて「日記」を作成できる
  - この単位を仮にDayと呼ぶことにする
  - `/@:user_name/:date`
  - 1個以上のBlockを束ねることでDayを作成できる
  - Dayは、トップページやタイムライン等の場所に表示されるイメージ
- Dayは「日記」としてひとまとまりのものとして表示される
  - が、その中のBlockごとにハッシュタグをつけることができる
  - Reblockすることで、既存のBlockを引用して自分のBlockとして投稿できる
- 書き手が書くタイミングと、読み手が読むタイミングは、微妙にズレる
  - 思い付いてすぐ投稿してすぐ拡散されて、といったリスキーな事態を避ける設計
- 書き手が公開できるのはDayだけなので、1日にひとつしか生成できず、流速が上がらない
