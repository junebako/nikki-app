# Nikki App

わたしの日記の書き方にマッチする、オリジナルの日記ウェブアプリケーション

## Features

- 投稿は小さい単位で行う
  - その単位をPieceと呼ぶ
  - Pieceは、𝕏のようなマイクロブログの1投稿のようなもの（テキスト、Markdown）
  - Pieceを投稿してもタイムラインのような場所には表示されない
- Pieceを束ねて「日記」を作成できる
  - この単位をDayと呼ぶ
  - `/@:user_name/:date`（例: `/@juneboku/2026-02-23`）
  - 1個以上のPieceを束ねることでDayを作成できる
  - Dayは、トップページやタイムライン等の場所に表示されるイメージ
- Dayは「日記」としてひとまとまりのものとして表示される
- 書き手が書くタイミングと、読み手が読むタイミングは、微妙にズレる
  - 思い付いてすぐ投稿してすぐ拡散されて、といったリスキーな事態を避ける設計
- 書き手が公開できるのはDayだけなので、1日にひとつしか生成できず、流速が上がらない

## Ideas

- `/@:user_name/:date.md` で日記本文をMarkdownで提供できるようにする（AIに食べさせる用）
- 日記のフィードを出力する
- ウェブブラウザ拡張機能を用意して、気軽にPieceを投稿できるようにする
